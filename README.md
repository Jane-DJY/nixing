# 逆行

一个用 p5.js 制作的 3:4 竖版动态视觉：一群像水彩笔触一样的蓝色鱼儿围绕最浅的中心深渊顺时针游动，画面带有旋涡、水雾和磨砂纸纹。隔较长一段时间，会有一条鱼脱离环流、从洞穴边缘游出去。

## 预览

直接打开 `index.html`，或用任意静态服务器托管本目录。

## 画面规格

- 画布尺寸：540 x 720
- 比例：3:4
- 适配：小红书竖版发布

## 操作

- 点击画面：重新播放
- 按 `S`：保存当前帧为 PNG

## 文件

- `index.html`：页面入口
- `styles.css`：画布居中与响应式比例
- `sketch.js`：p5.js 动画逻辑
